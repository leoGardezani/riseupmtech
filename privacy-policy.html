Política de Privacidade

Última atualização: 31 de março de 2026

A Rise Up MTech ("nós", "nosso" ou "empresa") está comprometida com a proteção da sua privacidade. Esta Política de Privacidade descreve como coletamos, usamos e protegemos suas informações ao usar nossos produtos e serviços, incluindo o Rise Up MTech ETL (Google Workspace Add-on) e o site riseupmtech.com.

Esta política se aplica especificamente ao uso do add-on Rise Up MTech ETL, que integra dados de plataformas de mídia ao Google Sheets mediante autorização explícita do usuário.

1. Informações que coletamos

Ao usar o Rise Up MTech ETL, podemos coletar as seguintes informações:

  • Endereço de e-mail — para identificação do usuário e gerenciamento de licenças
  • Tokens de acesso às plataformas de mídia — armazenados localmente no Google Apps Script Properties Service (criptografado pelo Google), nunca transmitidos a servidores da Rise Up MTech sem necessidade
  • Dados de uso — número de extrações realizadas, plataformas utilizadas e período consultado, usados exclusivamente para gerenciamento de licenças e melhoria do produto
  • Dados de mídia extraídos — métricas das suas campanhas publicitárias. Esses dados são escritos diretamente na sua planilha Google Sheets e não são armazenados em servidores da Rise Up MTech

2. Como usamos suas informações

Usamos suas informações exclusivamente para:

  • Autenticar sua identidade e verificar sua licença ativa
  • Executar as extrações de dados das plataformas autorizadas por você
  • Escrever os dados extraídos na sua planilha Google Sheets
  • Enviar comunicações relacionadas ao serviço (renovação de licença, atualizações)
  • Melhorar a qualidade e confiabilidade do produto

Não vendemos, alugamos ou compartilhamos suas informações pessoais com terceiros para fins de marketing.

3. Acesso a dados de terceiros

O Rise Up MTech ETL solicita acesso às APIs das seguintes plataformas mediante autorização explícita do usuário:

  • Google Ads (https://www.googleapis.com/auth/adwords)
  • YouTube Analytics (https://www.googleapis.com/auth/youtube.readonly)
  • Google Analytics / GA4 (https://www.googleapis.com/auth/analytics.readonly)
  • Google Sheets (https://www.googleapis.com/auth/spreadsheets)
  • Meta Ads API — mediante token fornecido pelo usuário
  • TikTok Ads API — mediante token fornecido pelo usuário
  • LinkedIn Ads API — mediante token fornecido pelo usuário
  • Pinterest Ads API — mediante token fornecido pelo usuário

O acesso é utilizado exclusivamente para leitura de dados de desempenho de campanhas e escrita na planilha do usuário. Não realizamos nenhuma operação de criação, modificação ou exclusão de campanhas nas plataformas.

Uso limitado: O uso de dados obtidos via APIs do Google se restringe ao propósito declarado do add-on (extração de métricas para Google Sheets) e está em conformidade com a Política de Dados do Usuário dos Serviços de API do Google, incluindo os requisitos de Uso Limitado.

4. Armazenamento e segurança

  • Tokens de acesso a plataformas de terceiros (Meta, TikTok, LinkedIn, Pinterest) são armazenados no Google Apps Script Properties Service, que é criptografado e gerenciado pelo Google
  • Dados de verificação de licença são transmitidos via HTTPS ao backend da Rise Up MTech hospedado no Google Cloud Run (região southamerica-east1)
  • Dados de campanhas extraídos são escritos diretamente na planilha do usuário e não passam por servidores da Rise Up MTech
  • Não armazenamos dados de campanhas em nossos servidores

5. Retenção de dados

Mantemos apenas os dados necessários para o funcionamento do serviço:

  • E-mail e status de licença: pelo período da assinatura + 90 dias
  • Logs de uso: até 12 meses
  • Tokens de plataformas: armazenados localmente pelo usuário e removidos ao desinstalar o add-on ou revogar o acesso

6. Seus direitos (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você tem direito a:

  • Confirmar a existência de tratamento de seus dados pessoais
  • Acessar seus dados pessoais
  • Corrigir dados incompletos, inexatos ou desatualizados
  • Solicitar a anonimização, bloqueio ou eliminação de dados desnecessários
  • Revogar o consentimento a qualquer momento
  • Solicitar a exclusão completa dos seus dados

Para exercer esses direitos, entre em contato: privacidade@riseupmtech.com

7. Revogação de acesso

Você pode revogar o acesso do add-on a qualquer momento:

  • Para dados Google: acesse myaccount.google.com/permissions e remova "Rise Up MTech ETL"
  • Para tokens de plataformas (Meta, TikTok, etc.): desinstale o add-on ou remova os tokens na aba Conexões

8. Cookies e rastreamento

O site riseupmtech.com não utiliza cookies de rastreamento ou analytics de terceiros. O add-on Rise Up MTech ETL não coleta dados de navegação.

9. Alterações nesta política

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos usuários ativos por e-mail sobre mudanças significativas. O uso continuado do serviço após a notificação constitui aceitação da política atualizada.

10. Contato

Para dúvidas, solicitações ou exercício de direitos relacionados à privacidade:

  • E-mail: privacidade@riseupmtech.com
  • Site: riseupmtech.com
